# Service Packages

## Target Market
€1M–€10M revenue Shopify stores. These stores have enough data to need BI but not enough resources to build it themselves.

---

## Packages

**For:** €1M–€3M revenue stores, single brand

**Includes:**
- Shopify pipeline setup (Airbyte → BigQuery → dbt → Looker)
- 3 core dashboards: Revenue & Margin, Customer LTV, Marketing Performance
- Manual COGS via CSV upload
- Monthly PDF report
- Email support (48h response)

**Delivery:** 3 weeks from kickoff
**Stack:** Airbyte Cloud, BigQuery, dbt Cloud, Looker Studio
